# snake game

This is a simple snake game made with python and pygame. Built using github copilot for demo.

## Steps to run

1. Create virtual environment
```bash
python -m venv venv
```

2. Activate virtual environment
```bash
source venv/Scripts/activate
```

3. install pygame
```bash
pip install pygame
```

4. Run the program
```bash
python main.py
```
